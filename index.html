Power Life｜每日營業日報使用說明

現場同仁依照「Key in → 確認 → 當天累積 → 結帳 → Google Sheet」流程操作。每一區塊都用深色框整理，方便手機與平板閱讀。

1整體流程

  1. 先在「Key in 資料」輸入收入或支出。
  2. 每一筆資料輸入完成後，按該列右側「確認」。
  3. 已確認資料會進入「當天累積 / 結帳」。
  4. 下班前檢查收入、支出、銷售人員與現金對帳。
  5. 確認無誤後按「當日結帳」送到 Google Sheet。

2Key in 資料

  • 收入明細：每收一筆款項，就新增一筆收入。
  • 支出明細：每支出一筆，就新增一筆支出。
  • 收入需選擇「銷售員 / 課程人員」。
  • 支出需選擇「支出 Key in 人員」。
  • 未按確認的資料不會進入結帳。

3當天累積 / 結帳

  • 這一頁會顯示已確認的收入與支出。
  • 結帳前必填：訪客數、新客數、會員來客數、票券使用。
  • 可以填 0，但不能空白。
  • 開店現金與閉店現金用於現金對帳。
  • 現金差異正常應為 0。

4結帳確認

  • 結帳前需填寫「結帳確認人員」。
  • 按「當日結帳」後，會跳出最後確認視窗。
  • 按「已確認無誤結帳」才會正式送到 Google Sheet。
  • 結帳完成後，今日資料會鎖定。

5區間總和查詢

  • 可以查詢單日或日期區間。
  • 起日與迄日選同一天，就是單日查詢。
  • 可用快速按鈕：當月、上個月、本週。
  • 可依銷售人員篩選營業額與收入明細。

6設定頁

  • 發票品項：控制收入明細的品項下拉選單。
  • 人員設定：銷售員 / 課程人員與支出 Key in 人員共用同一份名單。
  • 目前預設人員：YU、德明、Limit、Mia、宏。
  • 修改設定後會自動套用到新資料列。
